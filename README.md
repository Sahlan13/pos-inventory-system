# POS & Inventory Management System

## Description
A full-stack MERN (MongoDB, Express, React, Node) POS and Inventory Management App that allows admins and employees to manage products, sales, and stock efficiently. The frontend is built with React + Vite and Ant Design, while the backend uses Express.js and MongoDB Atlas.

## Live Demo
- Frontend: [https://pos-inventory-system-2.onrender.com](https://pos-inventory-system-2.onrender.com)  
- Backend: [https://pos-inventory-system-1.onrender.com](https://pos-inventory-system-1.onrender.com)

## Features
- User roles: Admin and Employee
- Add, edit, delete, and view products
- Search products by name or SKU
- Manage sales with cart functionality
- Real-time stock updates
- Mobile responsive design
